รับซอง TrueMoney

กรอกลิงก์ซองและเบอร์โทรศัพท์ที่ผูกกับบัญชี TrueMoney Wallet

🚀 เชื่อมต่อง่าย

เพียงส่ง POST request พร้อม voucher code และเบอร์มือถือ ก็สามารถรับเงินได้ทันที

⚡ รวดเร็ว

ประมวลผลรับ voucher ภายในไม่กี่วินาที พร้อม response ที่ชัดเจน

🔒 ปลอดภัย

รองรับ CORS ทุก origin ใช้งานได้จากทุก platform ปลอดภัยด้วย HTTPS

📊 Response ชัดเจน

ระบบส่ง response ครบถ้วน มีทุก status code ที่จำเป็น จัดการ error ได้ง่าย

API Documentation

📡 Endpoint

POST https://tw.oiio.in.th/api/:code/:mobile

📝 Parameters

:code - รหัส voucher (ตัดจาก URL gift.truemoney.com)

:mobile - เบอร์มือถือผู้รับ (เช่น 0812345678)

💡 ตัวอย่างการใช้งาน

curl -X POST https://tw.oiio.in.th/api/abc123xyz/0812345678

✅ Response Status Codes

SUCCESS - รับเงินสำเร็จ

TARGET_USER_REDEEMED - คุณรับซองนี้ไปแล้ว

VOUCHER_OUT_OF_STOCK - มีคนรับไปแล้ว

VOUCHER_EXPIRED - ซองวอเลทหมดอายุแล้ว

VOUCHER_NOT_FOUND - ไม่พบซองในระบบ

CANNOT_GET_OWN_VOUCHER - รับซองตัวเองไม่ได้

TARGET_USER_NOT_FOUND - ไม่พบเบอร์ในระบบ

INTERNAL_ERROR - ไม่พบซองในระบบ หรือ URL ผิด

📄 ตัวอย่าง Response

{ "status": { "code": "SUCCESS", "message": "Success" }, "data": { "voucher": { "amount_baht": "10.00", "member": 1, "status": "active" }, "my_ticket": { "amount_baht": "10.00", "mobile": "0812345678" } } }

🔧 Example Integration (JavaScript)

const redeemVoucher = async (code, mobile) => { const response = await fetch( `https://tw.oiio.in.th/api/${code}/${mobile}`, { method: 'POST' } ); const result = await response.json(); if (result.status.code === 'SUCCESS') { console.log('รับเงินสำเร็จ:', result.data.my_ticket.amount_baht); } else { console.error('Error:', result.status.message); } return result; }; // ใช้งาน redeemVoucher('abc123xyz', '0812345678');

🔧 Example Integration (PHP)

$code = 'abc123xyz'; $mobile = '0812345678'; $url = "https://tw.oiio.in.th/api/{$code}/{$mobile}"; $ch = curl_init($url); curl_setopt($ch, CURLOPT_POST, true); curl_setopt($ch, CURLOPT_RETURNTRANSFER, true); $response = curl_exec($ch); curl_close($ch); $result = json_decode($response, true); if ($result['status']['code'] === 'SUCCESS') { echo 'รับเงินสำเร็จ: ' . $result['data']['my_ticket']['amount_baht']; } else { echo 'Error: ' . $result['status']['message']; }
